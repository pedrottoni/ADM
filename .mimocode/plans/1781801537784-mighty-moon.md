# Fix: Restaurar controles da tabela "Gerenciar Histórico"

## Problema

A tabela `st.data_editor` na aba Financeiro (dentro do expander "Gerenciar Histórico de Transações") perdeu os controles de adicionar/remover linhas, busca e ordenação. A tabela renderiza corretamente e a edição de células funciona, mas a toolbar com botões +/- e search sumiu.

## Causa Raiz

CSS no `dashboard/static/cupertino.css` está escondendo/clipping os controles do data editor:

1. **`overflow: hidden` no `stDataFrame`** (linha 562) — corta a toolbar que fica no fundo do wrapper
2. **`overflow: hidden` no `stExpander`** (linha 514) — composta com o anterior, o editor dentro do expander sofre duplo clipping
3. **`.st-emotion-cache-1gulkj5 { display: none }`** (linha 638) — pode estar escondendo a toolbar inteira se o hash do Emotion coincidir com o container da toolbar no Streamlit 1.56.0

## Plano de Correção

### Arquivo: `dashboard/static/cupertino.css`

**1. Remover `overflow: hidden` do `stDataFrame` (linha 562)**
```css
/* ANTES */
div[data-testid="stDataFrame"] {
  border-radius: var(--dx-radius-m);
  border: 1px solid var(--dx-separator);
  overflow: hidden;
}

/* DEPOIS */
div[data-testid="stDataFrame"] {
  border-radius: var(--dx-radius-m);
  border: 1px solid var(--dx-separator);
}
```

**2. Trocar `overflow: hidden` por `overflow: visible` no `stExpander` (linha 514)**
```css
/* ANTES */
div[data-testid="stExpander"] {
  background: var(--dx-bg-secondary);
  border: 1px solid var(--dx-separator);
  border-radius: var(--dx-radius-m);
  overflow: hidden;
}

/* DEPOIS */
div[data-testid="stExpander"] {
  background: var(--dx-bg-secondary);
  border: 1px solid var(--dx-separator);
  border-radius: var(--dx-radius-m);
  overflow: visible;
}
```

**3. Remover a regra `.st-emotion-cache-1gulkj5 { display: none; }` (linha 638)**

Esta regra esconde um componente pelo hash do Emotion CSS-in-JS, que muda entre versões do Streamlit. É frágil e pode estar escondendo a toolbar. Remover completamente.

1. Rodar `streamlit run dashboard/app.py`
2. Navegar até aba "Financeiro" > sub-aba "Resumo Geral"
3. Abrir o expander "Gerenciar Histórico de Transações"
4. Confirmar que a toolbar aparece com:
   - Botão "+" para adicionar linha

   - Campo de busca/filtro
   - Ícones de ordenação nas colunas
5. Testar adicionar uma linha nova, deletar uma linha existente, e filtrar por texto
