# Fix: divisória vertical — pseudo-elemento na coluna

## Problema

`height: 100%` não funciona porque wrappers internos do Streamlit (`stElementContainer`) têm `height: auto` inline, quebrando a cadeia CSS. `100vh` quebra o layout. Pixel fixo não preenche a div.

## Solução

Voltar para 2 colunas e usar `::before` pseudo-elemento na 2ª coluna com `position: absolute; top: 0; bottom: 0`. Isso se estende automaticamente sem depender de `height: 100%` na cadeia.

### Arquivos

**`dashboard/tabs/financeiro.py`**
1. Reverter `st.columns([2, 0.05, 1])` → `st.columns([2, 1], gap="medium")`
2. Remover `col_div` e seu `st.html` (linhas ~164-168)

**`dashboard/static/cupertino.css`**
1. Adicionar regra para a 2ª coluna do chart card:

```css
[data-testid="stHorizontalBlock"]:has(.scrollable-chart-marker) > [data-testid="stColumn"]:nth-of-type(2) {
  position: relative !important;
}
[data-testid="stHorizontalBlock"]:has(.scrollable-chart-marker) > [data-testid="stColumn"]:nth-of-type(2)::before {
  content: "";
  position: absolute;
  left: -0.5rem;
  top: 0;
  bottom: 0;
  width: 1px;
  background: var(--dx-separator);
  pointer-events: none;
}
```

3. Posicionada no centro do gap entre as colunas
